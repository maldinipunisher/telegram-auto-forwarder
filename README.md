
# Telegram Auto Forwarder 

Telegram Auto Forwarder is a bot used to get update from another channel in telegram and then repost to target channel. 


## Installation

To deploy this project run

- Edit environtment.py then, 

```bash
  python setup.py 
  python main.py
```
Enjoy!
